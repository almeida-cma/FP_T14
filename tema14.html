📌 O que são Funções e Módulos?

Funções: Blocos de código reutilizáveis que executam uma tarefa específica.

Módulos: Arquivos .py que contêm funções e podem ser importados em outros programas.


  • def - Palavra usada para criar uma função
  • return - Retorna um valor da função
  • import - Importa um módulo
💡 Importante: Funções evitam repetição de código e deixam o programa mais organizado!

🔧 Funções Simples

Funções que executam uma tarefa sem retornar valor (apenas exibem).

Exemplo 1: Função de saudação

📄 funcao_saudacao.py 
# Funcao que exibe uma saudacao
def saudacao():
    print("Ola! Bem-vindo ao Python!")

# Chamando a funcao
saudacao()
saudacao()

Exemplo 2: Função que calcula e exibe o dobro

📄 funcao_dobro.py 
# Funcao que calcula e exibe o dobro
def calcular_dobro(numero):
    resultado = numero * 2
    print(f"O dobro de {numero} e {resultado}")

# Chamando a funcao
calcular_dobro(5)
calcular_dobro(10)
calcular_dobro(7)
💡 Dica: Use def para criar uma função e depois a chame pelo nome!

📥 Funções com Parâmetros e Return

Funções que recebem valores (parâmetros) e retornam um resultado.

Exemplo 1: Função que retorna a média

📄 funcao_media.py 
# Funcao que calcula e retorna a media
def calcular_media(nota1, nota2, nota3):
    media = (nota1 + nota2 + nota3) / 3
    return media

# Usando a funcao
resultado = calcular_media(7, 8, 9)
print(f"A media e: {resultado}")

# Chamando diretamente no print
print(f"Media: {calcular_media(10, 6, 8)}")

Exemplo 2: Função que retorna o quadrado

📄 funcao_quadrado.py 
# Funcao que retorna o quadrado de um numero
def quadrado(numero):
    return numero ** 2

# Testando a funcao
print(f"Quadrado de 4: {quadrado(4)}")
print(f"Quadrado de 7: {quadrado(7)}")
print(f"Quadrado de 10: {quadrado(10)}")
💡 Dica: Use return para enviar um resultado de volta para quem chamou a função!

📦 Criando e Usando Módulos

Módulos são arquivos .py que podem ser importados em outros programas.

Exemplo 1: Criando um módulo (salve como "meu_modulo.py")

📄 meu_modulo.py 
# Este e um modulo com funcoes uteis
def saudacao(nome):
    return f"Ola, {nome}!"

def dobro(numero):
    return numero * 2

def par_ou_impar(numero):
    if numero % 2 == 0:
        return "Par"
    else:
        return "Impar"

Exemplo 2: Usando o módulo (mesma pasta)

📄 usar_modulo.py 
# Importando o modulo criado
import meu_modulo

# Usando as funcoes do modulo
print(meu_modulo.saudacao("Ana"))
print(f"Dobro de 5: {meu_modulo.dobro(5)}")
print(f"10 e {meu_modulo.par_ou_impar(10)}")
print(f"7 e {meu_modulo.par_ou_impar(7)}")
💡 Dica: Para usar um módulo, ele deve estar na mesma pasta ou no caminho do Python!

🧮 Módulo math (Matemático)

O módulo math tem funções matemáticas prontas para usar.

Exemplo 1: Arredondamento (ceil e floor)

📄 math_ceil_floor.py 
# Importando o modulo math
import math

numero = 4.3

# ceil() - arredonda para cima
print(f"Numero: {numero}")
print(f"Arredondar para cima (ceil): {math.ceil(numero)}")

# floor() - arredonda para baixo
print(f"Arredondar para baixo (floor): {math.floor(numero)}")

Exemplo 2: Raiz quadrada e potência

📄 math_raiz.py 
# Importando o modulo math
import math

# sqrt() - raiz quadrada
numero = 25
print(f"Raiz quadrada de {numero}: {math.sqrt(numero)}")

# pow() - potência (mesmo que **)
print(f"2 elevado a 3: {math.pow(2, 3)}")
print(f"10 elevado a 2: {math.pow(10, 2)}")

Exemplo 3: Soma de lista com fsum

📄 math_fsum.py 
# Importando o modulo math
import math

# fsum() - soma precisa de numeros decimais
valores = [10.5, 20.3, 30.7, 15.2]
soma = math.fsum(valores)

print(f"Lista: {valores}")
print(f"Soma: {soma}")
💡 Dica: Use import math no início do programa para acessar todas essas funções!

💪 Exercícios sobre Funções

Exercício 1: Função que retorna o triplo

Crie uma função chamada triplo() que recebe um número e retorna o triplo dele.

📄 ex1_triplo.py 
# Exercicio 1: Funcao triplo
def triplo(numero):
    return numero * 3

# Testando
print(f"Triplo de 4: {triplo(4)}")
print(f"Triplo de 7: {triplo(7)}")
print(f"Triplo de 10: {triplo(10)}")

Exercício 2: Função de desconto

Crie uma função chamada aplicar_desconto() que recebe o preço e o percentual de desconto, e retorna o preço com desconto.

📄 ex2_desconto.py 
# Exercicio 2: Funcao de desconto
def aplicar_desconto(preco, percentual):
    desconto = preco * (percentual / 100)
    preco_final = preco - desconto
    return preco_final

# Testando
print(f"Preco com 10% de desconto: R$ {aplicar_desconto(100, 10):.2f}")
print(f"Preco com 20% de desconto: R$ {aplicar_desconto(50, 20):.2f}")

Exercício 3: Usando módulo math

Use o módulo math para calcular a raiz quadrada de 16, 25 e 49.

📄 ex3_raiz.py 
# Exercicio 3: Usando modulo math
import math

numeros = [16, 25, 49, 64, 81]

for num in numeros:
    print(f"Raiz quadrada de {num} = {math.sqrt(num)}")
